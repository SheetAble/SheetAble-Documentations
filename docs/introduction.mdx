# Introduction
SheetAble is an easy to use music sheet organiser and notation software available in your browser or on mobile devices. 

# Use
Currently the app can be installed on a home server on your computer or on iOS/Android.
<br>
You can upload sheet music to your "library", edit information for each file (such as composer name, a description, and an image), and create tags to organize your music.
<br>
# Future development
